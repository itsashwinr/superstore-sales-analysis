# Retail Sales Analysis Dashboard

## Project Overview

This project analyzes retail sales data from the Sample Superstore dataset using MySQL, Excel, and Tableau. The objective was to identify sales trends, profitability drivers, regional performance, customer segments, and the impact of discounts on business performance.

## Tools Used

- MySQL
- Tableau
- Microsoft Excel

## Dataset

- Sample Superstore Dataset
- Total Records: 9,994

## Business Objectives

- Measure overall sales and profitability.
- Identify top-performing regions, cities, and product categories.
- Analyze sales contribution by customer segment.
- Detect loss-making cities and sub-categories.
- Understand the relationship between discounts and profitability.
- Build an interactive dashboard for business reporting.

## Key Insights

- Technology generated the highest sales among all product categories.
- Profitability varied significantly across regions.
- A small number of cities contributed a large share of overall revenue.
- Several cities and sub-categories generated negative profits.
- Higher discount levels were associated with reduced profitability.
